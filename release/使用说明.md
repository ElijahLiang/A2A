# A2A — Avatar To Avatar 使用说明

> AI Agent 驱动的像素小镇社交游戏。  
> 本文件夹已包含完整程序，**无需安装 Python、Node.js 或任何依赖**，直接运行即可。

---

## 运行前准备：获取 API Key

游戏由 **DeepSeek AI** 驱动。运行前需要一个免费的 API Key：

1. 打开 https://platform.deepseek.com/
2. 注册账号 → 进入「API Keys」→ 点击「创建 API Key」
3. 复制生成的 Key（格式类似 `sk-xxxxxxxxxxxxxxxx`），下方步骤中会用到

## macOS 运行步骤

> 适用于 M1 / M2 / M3 / M4 芯片的 Mac（Apple Silicon）

### 第 1 步：打开终端

按 `Command + 空格`，搜索「终端」并打开。

### 第 2 步：进入本文件夹

将下方命令中的路径替换为你实际存放文件夹的路径：

```bash
cd /path/to/release文件夹
```

例如，如果文件夹在桌面：

```bash
cd ~/Desktop/release
```

### 第 3 步：赋予执行权限（只需做一次）

```bash
chmod +x A2A start.sh
```

### 第 4 步：解除系统安全限制（只需做一次）

macOS 会拦截从网络下载的程序，运行以下命令解除：

```bash
xattr -dr com.apple.quarantine ./A2A
```

### 第 5 步：启动游戏

将下方的 `你的Key` 替换为第一步获取的 API Key：

```bash
DEEPSEEK_API_KEY=你的Key ./start.sh
```

**示例：**

```bash
DEEPSEEK_API_KEY=sk-abc123456789 ./start.sh
```

### 第 6 步：打开游戏

浏览器会**自动弹出**。若未弹出，手动打开浏览器访问：

```
http://127.0.0.1:8000
```

### 停止游戏

在终端窗口按 `Control + C`，或直接关闭终端窗口。

## Windows 运行步骤

> ⚠️ 当前包内的 `A2A` 是 macOS 版本。  
> **Windows 用户需要联系发布者获取 `A2A.exe`（Windows 版本）。**

获取 Windows 版本的 `A2A.exe` 后，按以下步骤运行：

### 第 1 步：打开命令提示符

按 `Win + R`，输入 `cmd`，回车。

### 第 2 步：进入文件夹

```cmd
cd C:\path\to\release文件夹
```

例如，如果文件夹在桌面：

```cmd
cd %USERPROFILE%\Desktop\release
```

### 第 3 步：启动游戏

将 `你的Key` 替换为实际的 API Key：

```cmd
set DEEPSEEK_API_KEY=你的Key
A2A.exe
```

**PowerShell 用户：**

```powershell
$env:DEEPSEEK_API_KEY = "你的Key"
.\A2A.exe
```

### 第 4 步：打开游戏

浏览器会**自动弹出**。若未弹出，手动打开浏览器访问：

```
http://127.0.0.1:8000
```

### 停止游戏

关闭命令提示符窗口，或按 `Ctrl + C`。

> **Windows 安全提示**：首次运行时 Windows Defender 可能弹出警告，  
> 点击「更多信息」→「仍要运行」即可。

## 常见问题

**Q：启动后浏览器没有自动打开？**  
A：手动打开浏览器，访问 `http://127.0.0.1:8000`

**Q：提示"DEEPSEEK_API_KEY environment variable is not set"？**  
A：忘记设置 API Key 了。按照上方步骤重新运行，确保命令中包含 `DEEPSEEK_API_KEY=你的Key`

**Q：macOS 提示"无法打开，因为无法验证开发者"？**  
A：运行以下命令后再启动：
```bash
xattr -dr com.apple.quarantine ./A2A
```

**Q：8000 端口已被占用，无法启动？**  
A：指定其他端口：
```bash
DEEPSEEK_API_KEY=你的Key PORT=8080 ./A2A
# 然后访问 http://127.0.0.1:8080
```

**Q：Agent 说话很慢或没有对话气泡？**  
A：正常现象，AI 响应需要 2-5 秒。每隔 8 秒 Agent 会自动发起新一轮对话。
