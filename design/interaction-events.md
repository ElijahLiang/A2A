```

          │
          ▼

│  填写活动表单               │

│  · 时间（时间选择器，必填）   │

└─────────────┬─────────────┘
              │
              ▼
      检查 Token 余额
      ┌───────┴───────┐
      ▼               ▼
   余额充足         余额不足
      │               │
      ▼               ▼
   长按确认发布     提示：去完成活动
   (消耗Token)     或赛季任务赚取
      │
      ▼
   信封盖章动画 → 邮件飞出特效
      │
      ▼

      │
```

| 活动人数 | 消耗 Token |

| 3~5 人  | 2         |

## 七、奖励系统总览

### 7.1 Token 经济

```
         获 取                          消 耗
  ─────────────────              ─────────────────
  新用户注册       +3(初始)       发布2人活动    -1
  成功会面         +1             发布3~5人活动  -2
  合影留念         +2             发布不限人数   -3
  完成许愿池心愿   +1             投入心愿       -1
  赛季任务完成     +1~3           兑换限定家具   -5
  连续3天登录      +1             解锁特殊场馆   -3
  被爽约补偿       +1
```

### 7.2 集邮册系统

```
  邮戳生成规则：场馆 × 活动类型 × 对方身份 = 唯一邮戳

  邮戳等级：
  ┌──────────┬────────────────────────────────┐
  │ 普通（铜）│ 首次获得                        │
  │ 进阶（银）│ 同类型活动完成 3 次              │
  │ 稀有（金）│ 与同一人完成 5 次活动            │
  │ 隐藏（彩）│ 赛季限定 / 许愿池 / 隐藏猫咪活动 │
  └──────────┴────────────────────────────────┘

  集邮册解锁效果：
  · 集齐某场馆全系列 → 该场馆活动 Token 消耗减半
  · 集齐 10 枚不同邮戳 → 解锁小屋「邮戳展示墙」
  · 集齐赛季全系列   → 赛季限定称号 + 限定家具
```

### 7.3 像素小屋

```
  初始状态：一张桌子 + 一面空墙 + 一扇门

  ┌─────────────────────┬───────────────┬──────────────────┐
  │ 解锁条件             │ 获得物品       │ 功能效果          │
  ├─────────────────────┼───────────────┼──────────────────┤
  │ 首次会面             │ 椅子           │ 小屋可坐 2 人     │
  │ 与 5 个不同人会面     │ 窗户           │ 能看到广场动态    │
  │ 完成 3 个心愿        │ 猫窝           │ 猫咪来串门        │
  │ 合影 10 次           │ 照片墙         │ 展示像素合影      │
  │ 与同一人会面 5 次     │ 双人沙发       │ 好友专属动画      │
  │ 集齐体育馆邮戳       │ 运动角         │ 体育活动消耗 -50% │
  │ 赛季排名前 10%       │ 赛季限定装饰   │ 展示 + 称号       │
  │ Token 兑换           │ 各类家具       │ 纯装饰            │
  └─────────────────────┴───────────────┴──────────────────┘

  · 小屋半公开：好友可串门，陌生人看到外观缩略图
  · 小屋等级：家具数量决定等级（Lv1~Lv10），影响广场头衔显示
```

### 7.4 赛季系统

```
  赛季周期：每学期 1 赛季（约 4 个月）

  赛季内容：
  ┌──────────────────────────────────────────┐
  │  赛季主题（如「运动月」「读书季」）         │
  │  ├── 限定邮戳 ×5（赛季后绝版）             │
  │  ├── 限定猫咪 ×2（收入图鉴）               │
  │  ├── 限定家具 ×1（排名奖励）               │
  │  ├── 赛季任务线 ×10（完成 → 称号 + Token）  │
  │  └── 赛季排行榜（按会面次数 + 邮戳数）      │
  └──────────────────────────────────────────┘

  赛季结束时：
  · 排行榜快照保存
  · 限定内容锁定（已获得的保留，未获得不可再得）
  · Token 不清零，全部保留
```

```
  加好友触发条件：
  · 成功会面后，双方均可选择「加为好友」
  · 许愿池心愿达成后自动解锁

  好友功能：
  ┌───────────────┬──────────────────────────┐
  │ 直接私信       │ 不通过信件，直接留言      │
  │ 邀请参加活动   │ 跳过邮件，直接发送邀请卡  │
  │ 串门           │ 访问对方像素小屋          │
  │ 查看动态       │ 对方发布的活动 + 新邮戳    │
  │ 删除好友       │ 单方操作，对方收到通知     │
  └───────────────┴──────────────────────────┘
```

## 九、多人活动适配

```
  发布时选择人数 > 2：

  招募阶段：
  · 邮箱可同时接收多封申请信
  · 发布者逐一「接受」或「婉拒」
  · 凑齐人数 → 自动关闭招募
  · 48h 未凑齐 → 可选择「现有人数开始」或「延长 24h」

  会面阶段：
  · 到场 ≥ 2人 即可验证成功
  · 未到场者视为爽约，扣 Token

  合影阶段：
  · 多人合影：所有到场人排列在像素画中
  · 每人各获 Token +2

  邮戳：
  · 每位到场者两两之间各生成一枚邮戳
  · 3人到场 = 每人获得 2 枚邮戳
```

## 十、Agent 代聊规则

```
  触发条件：用户离线时收到私聊消息

  Agent 行为：
  ┌─────────────────────────────────────────────┐
  │ · 界面标注「AI 代回复中，真人上线后接管」       │
  │ · Agent 基于用户身份介绍 + 历史聊天风格回复     │
  │ · Agent 不可代做决策（不能接受/拒绝约定）       │
  │ · Agent 对话上限 5 轮，之后提示「等 TA 上线吧」 │
  │ · 用户上线后看到 Agent 对话记录，可继续或撤回   │
  └─────────────────────────────────────────────┘
```
